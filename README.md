Chatty
=====================

Simple chat application made with React

### Usage

Install dependencies and start the server from chatty-server, then start clientside app.

### Dependencies

* babel-core
* babel-loader
* babel-preset-es2015
* babel-preset-react
* css-loader
* node-sass
* sass-loader
* sockjs-client
* style-loader
* webpack
* webpack-dev-server
* react
* react-dom
* express
* ws
* uuid
